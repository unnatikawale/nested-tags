\# Nested Tags Tree — AIMonk Full Stack Assignment



A full stack web application for managing nested tag hierarchies.



\## Tech Stack

\- Frontend: React 18 + Vite

\- Backend: FastAPI (Python)

\- Database: SQLite



\## How to Run



\### Backend

cd backend

py -m venv venv

venv\\Scripts\\activate

pip install -r requirements.txt

py -m uvicorn main:app --reload



\### Frontend

cd frontend

npm install

npm run dev
